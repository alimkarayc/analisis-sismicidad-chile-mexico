# Comparative Seismic Activity Analysis

## Chile Central vs. Mexico–Central America

Statistical comparison of seismic activity between **Chile Central** and **Mexico–Central America** using earthquake records from the **USGS Earthquake Catalog (2000–2025)**.

This repository documents the complete workflow developed in **R**, including data preparation, exploratory analysis, spatial characterization, statistical inference and sensitivity analyses.

---

## Executive Summary

This academic project compares the seismic behaviour of **Chile Central** and **Mexico–Central America** using earthquake data reported by the USGS between 2000 and 2025.

The analysis includes:

- preprocessing and quality assessment of the earthquake catalogue;
- magnitude homogenization;
- exploratory and spatial data analysis;
- non-parametric statistical comparisons;
- correspondence analysis;
- count models;
- methodological sensitivity analysis.

Although developed as an academic project, this repository has been reorganized and documented as a reproducible data analysis portfolio.

---

## Objectives

- Compare seismic activity between Chile Central and Mexico–Central America.
- Characterize the spatial and temporal distribution of earthquakes.
- Evaluate differences in earthquake magnitude and focal depth.
- Explore tectonic profiles through correspondence analysis.
- Assess the robustness of the conclusions under different methodological decisions.
